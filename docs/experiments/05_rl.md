# 反転学習(reversal learning) — rl.py

これは基準達成で `r` と `nr` を入れ替える反転学習課題です。確率的反転学習の `prl.py` とは別物で、`rl.py` に `--sim` はありません。内部説明名は `touch_2stim_discrimination` 系です。

## 1. これは何か（目的）

左右に `r` と `nr` の2刺激を出し、正答率が閾値に達したら報酬ラベルを反転します。決まった正解を学習し、その後の反転に適応できるかを見る課題です。

## 2. いつ使うか（位置づけ）

[spsm.py](03_spsm.md) で2刺激判別が成立した後に使います。確率スケジュールを使う [prl.py](06_prl.md) へ進む前の、決定論的な反転学習です。全体の流れは [PROTOCOL.md](../PROTOCOL.md) を参照。

## 3. 前提条件

- 実行は `train/code/` ディレクトリ内で行う。共有モジュールが同じ場所にある前提。
- `--serial-port` と `--stim-dir` は必須。
- `--sim` と `--dry-run-ttl` はありません。タッチパネルと Arduino TTL が使える実機環境で起動します。
- `--stim-dir` には `stim_NN_r.png` と `stim_NN_nr.png` のペアを置きます。

## 4. 起動コマンド例

本番最小形:

```bash
cd train/code
python rl.py --serial-port /dev/ttyACM0 --stim-dir ../visual_stimuli
```

反転を有効にする例:

```bash
cd train/code
python rl.py --serial-port /dev/ttyACM0 --stim-dir ../visual_stimuli --reversals-per-set 1
```

共通引数は [共通引数リファレンス](../_common_args.md) を参照。

## 5. この実験に固有の主要引数

| 引数 | 既定値 / choices | 説明 |
|---|---|---|
| `--stim-dir` | required | `stim_NN_r.png` / `stim_NN_nr.png` のペアが入ったフォルダ。 |
| `--sliding-n` | `20` | 正答率判定の試行窓。 |
| `--acc-threshold` | `0.8` | 逆転/昇格判定の閾値。help では「以上」で判定。 |
| `--correction-mode` | off | 失敗後、成功まで前試行と同じ左右配置を繰り返す。 |
| `--exclude-correction-from-acc` | off | 正答率判定からコレクショントライアルを除外する。 |
| `--reversals-per-set` | `0` | 各刺激セットでの逆転回数。`0` は逆転なし、`1` 以上で閾値到達ごとに `r` / `nr` を入れ替える。 |

## 6. 出力CSV

- 保存先: `--out-dir`。既定値は `logs/`。
- ファイル名 prefix: `touch_2stim_log_`
- 共通列は [CSV_FORMAT.md](../CSV_FORMAT.md) を参照。

重要列:

| 列 | 説明 |
|---|---|
| `stim_set` | 刺激セット番号ラベル。 |
| `left_label`, `right_label` | 左右に出た `r` / `nr` ラベル。 |
| `target_label` | 現在の報酬ラベル。 |
| `reversal_count_in_set` | セット内の反転回数。 |
| `reversals_per_set` | 起動時に指定した反転回数設定。 |
| `reversal_event`, `reversal_idx`, `reversal_trigger_acc` | 反転発生と、そのときの判定情報。 |
| `sliding_correct`, `sliding_acc` | 判定窓内の成績。 |
| `is_correction_trial` | コレクショントライアルかどうか。 |

## 7. 典型パラメータ

導入用（反転なし、既定の判定窓）:

```bash
cd train/code
python rl.py --serial-port /dev/ttyACM0 --stim-dir ../visual_stimuli --reversals-per-set 0 --sliding-n 20 --acc-threshold 0.8 --show-box
```

本番用（1回反転、タッチ専用）:

```bash
cd train/code
python rl.py --serial-port /dev/ttyACM0 --stim-dir ../visual_stimuli --reversals-per-set 1 --fullscreen --touch-only --kiosk
```

## 8. 終了方法

- `train/code/` に `STOP` ファイルを置く。
- ウィンドウを閉じる。
- `ESC` または `q` を押す。

## 9. よくある失敗

- `prl.py` と間違えて `--sim` を付けても、`rl.py` には存在しないため argparse エラーになります。
- `--stim-dir` に `stim_NN_r.png` / `stim_NN_nr.png` のペアがないと起動できません。
- `--reversals-per-set 0` は反転なしです。反転を見たい場合は `1` 以上を指定します。
- 横断的な問題は [TROUBLESHOOTING.md](../TROUBLESHOOTING.md) を参照。
